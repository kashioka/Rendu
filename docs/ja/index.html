AIが作るドキュメント、きれいに読めるビューア

Markdown（.md）はAIが出力する標準的なドキュメント形式です アプリを起動して、フォルダかファイルを選ぶだけ 図表・表・コードもきれいに整形して表示します
macOS 版ダウンロード Windows / その他

または: brew install --cask kashioka/tap/rendu

macOS (Apple Silicon) Windows (x64)
Windows をお使いの方へ: .exe インストーラのダウンロード時に「一般的にダウンロードされていません」という警告が表示されることがあります。 Windows コード署名証明書を未取得のため表示されるもので、アプリ自体に問題はありません。 「保持する」をクリックして進めてください。 ソースコードは GitHub で公開しています。

機能

ファイルツリー

フォルダ内の .md ファイルをツリー表示。クリックですぐプレビュー。

Mermaid 図表

フローチャート、シーケンス図、ER図などをインライン描画。

PDF 出力

表示中のページをワンクリックでA4 PDFにエクスポート。

テーマカスタマイズ

ダーク・ライトテーマ。背景色、文字色、コードブロック色を自由に変更。

ドキュメントアウトライン

見出し一覧をサイドバーに表示。クリックでジャンプ。

GFM 完全対応

テーブル、タスクリスト、取り消し線、シンタックスハイライト対応。
