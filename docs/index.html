Rendu is French for "a finished rendering" or "something delivered." Just as architects create a rendu to convey their design intent, Rendu transforms your Markdown blueprints into an intuitive, beautifully rendered form.

For Everyone

Just open a folder and start reading.

📂

Folder browsing

Open any folder to see a tree of .md files. Click to read. Drag and drop works too.

🎨

Light & dark themes

Switch themes or customize every color. Looks great at any time of day.

📄

PDF export

One-click export to a clean A4 PDF. Share rendered docs with anyone.

📑

Document outline

Heading sidebar with click-to-jump navigation. Find your place instantly.

📋

Copy code blocks

Click the copy button on any code block. Paste directly into your project.

🕒

Recent files

Quickly jump back to files you recently opened. Your history is always saved.

For Engineers

The tools you expect, built in from day one.

📐

Mermaid diagrams

Flowcharts, sequence diagrams, ER diagrams, Gantt charts — all rendered inline.

💡

Syntax highlighting

Automatic language detection and highlighting for code blocks. 190+ languages.

📝

Full GFM support

Tables, task lists, strikethrough, autolinks — GitHub Flavored Markdown, fully supported.

⌨

CLI

rendu file.md opens and renders from the terminal. Install via Homebrew on macOS.

🔗

File associations

Double-click any .md file in Finder or Explorer to open it directly in Rendu.

⚙

Native & fast

Built with Tauri 2.0 + Rust. Under 15 MB on disk. No Electron bloat.
